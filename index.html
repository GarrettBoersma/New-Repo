Course Catalog Description:

This course will cover the creation of web pages and sites using HTML, CSS, Javascript and grapical applicatons. Networked multimedia distribution technologies are also explored. The design of effective Web site including age layout, user interface design graphic design, content flow and site structure as well as management of web site resources including internet management and design considerations are addressed. Students design and create a major web site with multiple pages and cross-linked structures. Prereqisites:None Credit:2-2-3 Semester Hours
